DMEdit

A fast text editor that handles files of any size

Get Started What's New

Large File Support

Open and smoothly scroll through multi-gigabyte files. Memory usage stays low regardless of file size.

Tabbed Editing

Work with multiple documents in tabs. Each tab preserves its scroll position, selection, and undo history.

Find & Replace

Search with literal text, wildcards, or regular expressions. Live match count and highlighting as you type.

Keyboard Profiles

Choose from Default, VS Code, Visual Studio, JetBrains, Eclipse, or Emacs key bindings. Customize individual shortcuts.

Print & PDF Export

Print documents with configurable margins and paper sizes. Export directly to PDF. Progress tracking for large documents.

Smart Editing

Auto-indent, smart backspace, column selection with multi-cursor editing, clipboard history, and case transforms.
