What's New

User-facing highlights from each release.

Version 0.6 April 2026

Printing

  • Print your documents — full print support with configurable printer, paper size, orientation, and margins. DMEdit remembers your print settings between sessions.
  • Progress tracking — large documents show a progress dialog with page count and estimated time remaining. You can cancel at any time.
  • Save as PDF — export any document directly to a PDF file.

Smart Editing

  • Auto-indent on Enter — new lines automatically inherit the leading whitespace from the current line.
  • Smart Backspace — when the caret is in leading whitespace, Backspace jumps to the previous indent stop instead of deleting one space at a time.
  • Smart Home key — pressing Home toggles between the first non-whitespace character and column 1.
  • Trailing whitespace cleanup — pressing Enter removes trailing spaces from the line you just left.

Display Improvements

  • Wrap indicators — a small arrow glyph shows where a line wraps, making it clear which rows are continuations of a wrapped line.
  • Hanging indent — wrapped continuation rows are indented by half an indent level, so the start of each logical line is visually distinct.
  • Fast monospace rendering — a new rendering path for monospace fonts that is significantly faster and enables hanging indent. Ligatures can be re-enabled in Settings if preferred.

Large File Handling

  • Character-wrap mode — files with extremely long lines (such as minified code or single-line JSON) now wrap at fixed character widths with optimized scroll math. The editor stays responsive regardless of line length.

Reliability

  • Surrogate pair safety — Backspace, Delete, and clipboard operations correctly handle emoji and other characters that use surrogate pairs.
  • Binary file handling — opening a file that contains binary control characters no longer causes crashes during scrolling.
  • Scroll accuracy — fixed several issues where arrow-key scrolling at the viewport edge could jump or lose sync with the scrollbar.

Version 0.5 February – March 2026

Tabbed Document Interface

  • Multiple documents — open files in tabs, each with its own scroll position, selection, and undo history.
  • Tab switching — use Ctrl+Tab and Ctrl+Shift+Tab, or click a tab to switch.
  • Unsaved changes indicator — a dot appears on tabs with unsaved changes. It clears when you save or undo back to the saved state.

Status Bar

  • Interactive status bar — shows line, column, encoding, line endings, and indent style. Click elements to access related actions (Go To Line, encoding conversion, indent conversion).
  • Indentation detection — DMEdit auto-detects whether a file uses spaces or tabs when it's opened.
  • Line ending and indent conversion — convert between LF/CRLF or spaces/tabs with one click. Mixed-ending tooltips show the count of each type.

Find Bar

  • Search with highlighting — all matches are highlighted as you type, with a live match count inside the search box.
  • Search modes — toggle between literal text, wildcard patterns, and regular expressions.
  • Find Next/Previous Selection — jump between occurrences of the selected text without opening the find bar.
  • Incremental search — type-to-search mode that jumps to matches as you type.
  • Resizable find bar — drag the left edge to make the search box wider or narrower.

Keyboard Customization

  • Six keyboard profiles — Default, VS Code, Visual Studio, JetBrains, Eclipse, and Emacs.
  • 55+ commands — a centralized command system with customizable bindings for every action.
  • Chord shortcuts — two-key combinations for less frequent commands (for example, Ctrl+E, Ctrl+W for wrap toggle).

Editing

  • Case transforms — convert selected text to UPPER CASE, lower case, or Proper Case.
  • Move line up/down — rearrange lines with Alt+Up/Down.
  • Delete line — remove the entire current line with Ctrl+Y.
  • Delete word — Ctrl+Backspace and Ctrl+Delete delete one word at a time.
  • Undo restores selection — undo now restores your exact selection (anchor position and direction), not just the caret.
  • Revert file — discard all unsaved changes and reload from disk.

Navigation

  • Go To Line — jump to a specific line or line:column with Ctrl+G.
  • Command palette — search and run any command with F1.
  • Middle-button scroll — middle-click and drag to scroll freely in any direction.

Display

  • Line numbers — a gutter on the left showing line numbers, with auto-adjusting width.
  • Column guide — a vertical line at the wrap column, visible whether wrapping is on or off.
  • Horizontal scrollbar — when word wrap is off, a horizontal scrollbar appears for long lines.
  • Window state persistence — window size, position, and maximized state are remembered between sessions.
