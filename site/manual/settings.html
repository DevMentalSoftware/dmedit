Settings

All configurable preferences, organized by category.

Open Settings from the Edit menu or through the command palette (F1). Settings are saved automatically and take effect immediately. They persist between sessions.

Display

SettingDefaultDescription
Theme System Choose between Light, Dark, or System (follows your OS setting).
Use Wrap Column On When enabled, lines wrap at the Wrap Column limit instead of the viewport edge.
Wrap Column 100 Column at which lines wrap when Use Wrap Column is enabled. Range: 1 – 10,000.
Show Wrap Symbol On Show a small arrow glyph at the wrap point for lines that wrap to the next row.
Hanging Indent On Indent wrapped continuation rows by half an indent level. Makes it easy to see where logical lines begin.
Fast Text Layout On Use the fast monospace rendering path. Faster and enables hanging indent, but disables font ligatures. Turn off if you prefer ligatures.
Caret Width 1.0 Width of the text caret in pixels. Range: 1.0 – 2.5, in 0.5 increments.
Bright Selection Off Use a brighter, more visible selection highlight. Useful on monitors with limited contrast.

Editor

SettingDefaultDescription
Indent Width 4 Number of columns per indent level. Also controls the display width of tab characters. Range: 1 – 16.
Undo Coalesce Timer 1000 ms Idle time before consecutive edits are committed as a single undo entry. Higher values group more keystrokes together. Range: 100 – 10,000 ms.
Expand Selection Mode Word Controls how Expand Selection (Ctrl+Shift+W) grows the selection. "Word" starts at whitespace boundaries; "SubwordFirst" starts at camelCase/underscore boundaries.
Auto-Reload External Changes Off Automatically reload files modified by other programs, as long as the tab has no unsaved edits.
Backup on Save Off Keep a .bak copy of the previous version when saving.
Tail File Off When a file is reloaded and the editor is scrolled to the bottom, automatically scroll to show new content. Useful for monitoring log files.

Font

The font section (within Settings) lets you configure:

  • Font Family — choose any installed font. The default is Cascadia Code with Consolas and Courier New as fallbacks.
  • Font Size — set the base font size in points (default: 11).
  • Zoom — set the zoom percentage (default: 100%). Range: 10% – 800%. Zoom is independent of font size and can also be changed with Ctrl+PageUp/PageDown.

A live preview shows sample text in your chosen font.

Scrollbar

SettingDefaultDescription
Outer Thumb Scroll Rate 2.0 Multiplier for the outer-thumb fixed scroll speed. Higher values scan through the document faster. Range: 0.1 – 20.0.

Advanced

SettingDefaultDescription
Recent File Count 10 Number of files shown in the File > Recent list. Range: 0 – 50.
Auto-Update On Download updates silently on startup and show a restart indicator in the status bar. When off, updates are still detected but require a manual download from Settings.
Char Wrap File Size (KB) 50 Files larger than this that also contain very long lines automatically switch to character-wrapping mode. See Large Files.
Distribute Multi-Line Paste in Column Mode On When pasting into a column selection, distribute clipboard lines across cursors when counts match. When off, the entire clipboard is pasted at every cursor. See Column Selection.

Commands

SettingDefaultDescription
Chord Timeout 3000 ms Time before a pending chord prefix is cancelled. When using a two-key chord shortcut, press the second key within this timeout. Range: 500 – 10,000 ms.

Settings File

Settings are stored in a settings.json file in your user data directory. Some advanced settings that aren't shown in the Settings UI can be edited directly in this file. Changes are picked up when DMEdit restarts.
