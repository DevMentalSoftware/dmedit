Status Bar

The interactive status bar at the bottom of the editor window.

The status bar shows information about the current document and your position in it. Many of the elements are clickable, giving you quick access to related actions.

Toggle the status bar from View > Status Bar.

Position Display

The left side of the status bar shows:

  • Ln — the current line number.
  • Ch — the current column number.
  • Selection count — when text is selected, the number of selected characters is shown.
  • Total lines — the total number of lines in the document.

Click the Ln / Ch display to open the Go To Line dialog.

Encoding

Shows the file's character encoding (for example, UTF-8). Click to see encoding options.

Line Endings

Shows the line ending style used in the document:

  • LF — Unix-style line feed (used on Linux and macOS).
  • CRLF — Windows-style carriage return + line feed.

Click to convert between line ending styles. If the document has mixed line endings, a tooltip shows the count of each type. Conversion is a single undo operation.

Indent Style

Shows the current indentation style:

  • Spaces — indentation uses space characters.
  • Tabs — indentation uses tab characters.

When you open a file, DMEdit detects the indent style automatically. Click the indicator to convert all indentation in the document between spaces and tabs. Conversion is a single undo operation.

Insert and Overwrite

Shows the current editing mode:

  • INS — insert mode (typed characters push existing text to the right).
  • OVR — overwrite mode (typed characters replace the character under the caret).

Click to toggle between modes. You can also press Insert on the keyboard.

Tail Mode

When tail mode is active, a Tail indicator appears in the status bar. Tail mode keeps the editor scrolled to the bottom of the file, so when the file is reloaded with new content (for example, a growing log file), you automatically see the latest lines.

Click the indicator to toggle tail mode on or off. Tail mode only takes effect when the file is reloaded from disk (either manually or via auto-reload).

See Settings for more on auto-reload and tail mode.

Update Notification

When a new version of DMEdit is available and has been downloaded, a notification appears in the status bar. Click it to restart and apply the update.

Automatic updates can be turned on or off in Settings.
