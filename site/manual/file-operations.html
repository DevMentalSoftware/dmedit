File Operations

Creating, opening, saving, printing, and managing files.

Creating a New File

Press Ctrl+N or choose File > New to create a blank document. The new file opens in its own tab with the title "[New Document]" until you save it.

Opening Files

Press Ctrl+O or choose File > Open to open one or more files. You can select multiple files in the dialog to open them all at once. Each file opens in its own tab.

You can also drag and drop files from your file manager onto the editor window.

If you open a file that is already open in another tab, DMEdit switches to the existing tab instead of opening a duplicate.

Recent Files

The File menu shows a list of recently opened files at the bottom. Click a file to reopen it. You can clear this list from the menu if needed. The number of recent files shown is configurable in Settings (default: 10).

Saving

ActionShortcutDescription
Save Ctrl+S Save the current file. If it's a new document, prompts for a file name.
Save As Ctrl+Alt+S Save the current file under a new name or location.
Save All Ctrl+Shift+S Save all open files that have unsaved changes.

A dot appears in the tab header when a document has unsaved changes. It disappears once you save. If you undo all changes back to the last saved state, the dot also disappears.

Backup on Save

When enabled in Settings, DMEdit keeps a .bak copy of the previous version each time you save. This gives you a safety net if you need to recover the last saved version.

Reverting and Reloading

ActionDescription
Revert File Discards all unsaved changes and reloads the file from disk. Your undo history is cleared.
Reload File Reloads the file from disk. Available when the file has been modified externally.

Auto-Reload External Changes

When enabled in Settings, files that are modified by another program are automatically reloaded, as long as you don't have unsaved edits in that tab. If the tab has unsaved changes, a conflict icon appears in the tab instead, and you can decide whether to reload or keep your version.

Read-Only Mode

Use File > Toggle Read Only to prevent accidental edits. When a file is read-only, a badge appears on its tab and all editing commands are disabled. Toggle it again to resume editing.

Files that are read-only on disk also open in read-only mode automatically.

Closing Files

ActionShortcutDescription
Close Ctrl+F4 Close the current tab. Prompts to save if there are unsaved changes.
Close All Ctrl+Shift+F4 Close all open tabs. Prompts to save any tabs with unsaved changes.

Printing

Press Ctrl+P or choose File > Print to print the current document. The print dialog lets you choose:

  • Printer
  • Paper size
  • Orientation (portrait or landscape)
  • Margins

For large documents, a progress dialog shows measurement and page rendering progress with an estimated time remaining. You can cancel printing at any time.

DMEdit remembers your print settings between sessions.

Save as PDF

Choose File > Save as PDF to export the current document as a PDF file. This uses the same layout engine as printing, including hanging indent for wrapped lines.

Exiting

Press Alt+F4 or choose File > Exit to close DMEdit. If any tabs have unsaved changes, you'll be prompted to save them before closing.
