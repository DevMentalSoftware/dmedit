Smart Home

Pressing Home toggles between two positions:

  • The first non-whitespace character on the line.
  • Column 1 (the very beginning of the line).

If the caret is anywhere in the middle of the line, the first press of Home moves to the first non-whitespace character. Press Home again to jump to column 1, and again to jump back.

Go To Line

Press Ctrl+G to open the Go To Line dialog. Enter a line number and press Enter to jump to that line. You can also enter line:column to jump to a specific position (for example, 150:20).

You can also open this dialog by clicking the Ln / Ch indicator in the Status Bar.

Command Palette

Press F1 to open the command palette. Start typing to filter the list of all available commands. Press Enter to run the highlighted command, or Escape to close the palette.

The command palette is the quickest way to access any command, especially those that don't have a keyboard shortcut assigned.

Column Selection

Column selection (also called block selection) lets you select a rectangular block of text and edit multiple lines simultaneously.

Starting a Column Selection

Hold Alt and click to start a column selection, then drag to extend it. You can also use keyboard shortcuts:

ActionShortcut
Column select upAlt+Shift+Up
Column select downAlt+Shift+Down
Column select leftAlt+Shift+Left
Column select rightAlt+Shift+Right

Editing with Column Selection

When a column selection is active, you have a cursor on every selected line. Anything you type is inserted at all cursor positions simultaneously. This is useful for:

  • Adding the same prefix or suffix to many lines at once.
  • Deleting a column of characters across multiple lines.
  • Replacing a rectangular block of text.

Pasting into a Column Selection

When pasting into a column selection with multiple cursors:

  • Distribute mode (default) — if your clipboard has the same number of lines as cursors, each line is pasted at the corresponding cursor. Extra lines are dropped.
  • Broadcast mode — the entire clipboard contents are pasted at every cursor position.

You can switch between distribute and broadcast mode in Settings.
