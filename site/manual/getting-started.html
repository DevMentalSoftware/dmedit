Getting Started

A quick tour of the DMEdit interface and basic operations.

The Editor Window

When you open DMEdit, you'll see these main areas:

  • Tab bar — across the top, showing one tab for each open document. Click a tab to switch to it.
  • Line number gutter — on the left side, showing line numbers for the current document.
  • Editor area — the main text area where you read and edit your document.
  • Scroll bar — on the right edge. DMEdit uses a dual-zone scroll bar: the inner thumb jumps to a position proportionally, while the outer thumb scrolls at a fixed speed for scanning through large files.
  • Status bar — along the bottom, showing your current line and column, encoding, line endings, and indent style. Many of these elements are clickable. See Status Bar for details.

Opening a File

There are several ways to open a file:

  • Press Ctrl+O or use File > Open to browse for a file.
  • Drag a file from your file manager and drop it onto the editor.
  • Choose a recently opened file from File > Recent Files.
  • Double-click a file in Windows Explorer if DMEdit is set as the default editor for that file type.

Each file opens in its own tab. If you open a file that's already open, DMEdit switches to the existing tab rather than opening a duplicate.

Basic Editing

Editing works as you'd expect from any text editor:

  • Click to place the caret, then type to insert text.
  • Press Enter to start a new line. DMEdit auto-indents the new line to match the current one.
  • Press Backspace to delete backward, Delete to delete forward.
  • Press Tab to indent. The indent width and style (spaces or tabs) are shown in the status bar and can be changed in Settings.

Undo and Redo

Press Ctrl+Z to undo and Ctrl+Shift+Z to redo. DMEdit groups consecutive keystrokes into a single undo entry, so undoing jumps back over a natural chunk of typing rather than one character at a time.

Saving

Press Ctrl+S to save. If the document hasn't been saved before, you'll be prompted to choose a file name. A dot appears in the tab when a document has unsaved changes, and disappears once you save.

Finding Text

Press Ctrl+F to open the find bar at the top of the editor. Type your search term and press F3 to find the next match or Shift+F3 for the previous match. The total match count is displayed inside the search box. See Search & Replace for full details.

Navigating by Line Number

Press Ctrl+G to open the Go To Line dialog. Type a line number to jump directly to it. You can also type line:column to jump to a specific position (for example, 42:10 to go to line 42, column 10).

Command Palette

Press F1 to open the command palette. This gives you quick access to every command in DMEdit by typing part of its name. It's the fastest way to find a command whose shortcut you don't remember.
