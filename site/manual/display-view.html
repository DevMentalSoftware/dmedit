Display & View

Line numbers, whitespace, word wrap, zoom, and other visual options.

Line Numbers

Toggle line numbers from View > Line Numbers. When enabled, a gutter on the left side of the editor shows the line number for each line. The gutter width adjusts automatically based on the number of digits needed.

Whitespace Glyphs

Toggle whitespace visibility with the chord shortcut Ctrl+R, Ctrl+W (press Ctrl+R, then press Ctrl+W within 3 seconds), or from View > Whitespace.

When enabled, spaces, tabs, and non-breaking spaces are shown as visible glyphs. This is useful for spotting mixed indentation or trailing whitespace.

Word Wrap

Toggle word wrap with the chord shortcut Ctrl+E, Ctrl+W, or from View > Wrap Lines.

When wrapping is enabled, long lines are soft-wrapped to fit within the editor viewport. The actual line content isn't changed — wrapping is purely visual.

Wrap Column

By default, DMEdit wraps lines at a column limit (default: 100) rather than the viewport edge. This can be configured in Settings:

  • Use Wrap Column — when enabled, lines wrap at the column limit or the viewport edge, whichever is narrower. When disabled, lines wrap at the viewport edge only.
  • Wrap Column — the column number where wrapping occurs (default: 100).

Column Guide

When Use Wrap Column is enabled, a thin vertical line is drawn at the wrap column. This provides a visual reference for line length even when wrapping is turned off.

Hanging Indent

When wrapping is on, continuation rows (the second and subsequent visual rows of a wrapped line) are indented by half of one indent level. This makes it easy to distinguish where a new logical line begins versus where a previous line continues.

Hanging indent can be toggled in Settings.

Wrap Symbol

When wrapping is on, a small arrow glyph appears at the wrap column for lines that wrap to the next row. This can be toggled in Settings.

Horizontal Scrolling

When word wrap is off, a horizontal scrollbar appears if any visible line extends beyond the viewport edge. Use it to scroll left and right through long lines.

Zoom

Zoom ranges from 10% to 800%. The zoom level is independent of font size — both can be adjusted separately. The current zoom level is saved and restored between sessions.

Theme

DMEdit supports light, dark, and system themes. The theme can be changed in Settings. When set to "System", DMEdit follows your operating system's light/dark mode preference.

Fast Text Layout

By default, DMEdit uses a fast rendering path for monospace fonts. This is significantly faster and enables hanging indent, but disables font ligatures (where characters like => are displayed as a single combined glyph).

If you prefer ligatures, you can turn off Fast Text Layout in Settings. This falls back to standard text rendering, which is slower but supports full font shaping.

Bright Selection

If you find the default selection highlight hard to see on your monitor, enable Bright Selection in Settings for a higher-contrast highlight color.
