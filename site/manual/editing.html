Editing

Undo, clipboard, line operations, indentation, and case transforms.

Undo and Redo

ActionShortcut
UndoCtrl+Z
RedoCtrl+Shift+Z

DMEdit groups consecutive keystrokes into a single undo entry. When you pause typing, the current group is committed and the next keystrokes start a new group. This means undo jumps back over a natural chunk of editing rather than undoing one character at a time.

The grouping delay is configurable in Settings (default: 1000 ms).

Undo also restores your exact selection — the anchor position and direction are preserved, so after undoing you see the same selection you had before the edit.

Clipboard

ActionShortcut
CutCtrl+X
CopyCtrl+C
PasteCtrl+V

Paste More (Clipboard Cycling)

Press Ctrl+Shift+V to paste the previous clipboard entry. Each press cycles back through your clipboard history while you keep Ctrl held. Release Ctrl to confirm the pasted text.

This lets you quickly access recently copied text without opening a separate dialog.

Clipboard Ring

DMEdit keeps a history of your recent clipboard entries (up to 10 by default). Use Edit > Clipboard Ring to open a popup showing all entries, where you can click one to paste it.

Line Operations

ActionShortcutDescription
Move line up Alt+Up Swap the current line (or selected lines) with the line above.
Move line down Alt+Down Swap the current line (or selected lines) with the line below.
Duplicate line — Insert a copy of the current line below it. Available from the Edit menu or command palette.
Insert line below — Insert a blank line below the current line and move the caret to it.
Insert line above — Insert a blank line above the current line and move the caret to it.
Tip: Commands without a default shortcut can be found in the command palette (F1) or assigned a custom shortcut in Keyboard Shortcuts.

Indentation

Auto-Indent

When you press Enter, the new line automatically inherits the leading whitespace from the current line. DMEdit also removes trailing whitespace from the line you just left.

Smart Backspace

When the caret is in the leading whitespace of a line, pressing Backspace jumps back to the previous indent stop rather than deleting one space at a time. This works when the indent style is set to spaces.

Indent and Outdent

ActionShortcut
Indent (or smart indent)Tab
OutdentShift+Tab

When text is selected, Tab indents all selected lines and Shift+Tab removes one level of indentation. When nothing is selected, Tab inserts spaces or a tab character depending on the current indent style.

Converting Indentation

You can convert between spaces and tabs for the entire document. Click the indent style indicator in the Status Bar to access conversion options, or use the Edit menu.

Insert and Overwrite Modes

Press Insert to toggle between insert mode (new text pushes existing text to the right) and overwrite mode (new text replaces the character under the caret). The current mode is shown in the Status Bar and can be toggled by clicking it there.

Expand Selection

Press Ctrl+Shift+W to progressively expand the selection. Each press selects a larger scope: subword boundary, word, line, then the entire document. The expansion mode (subword-first or word-first) is configurable in Settings.
