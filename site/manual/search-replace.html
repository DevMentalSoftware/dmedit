Search & Replace

Finding and replacing text with literal, wildcard, or regex matching.

Opening the Find Bar

Press Ctrl+F to open the find bar at the top of the editor. The find bar has a search box, match options, and navigation buttons.

Press Ctrl+H to open the find bar in replace mode, which adds a replacement text box and replace buttons.

Press Escape to close the find bar and return focus to the editor.

Finding Text

ActionShortcut
Find next matchF3
Find previous matchShift+F3
Find next using selectionCtrl+F3
Find previous using selectionCtrl+Shift+F3

As you type in the search box, all matches are highlighted in the editor and the total match count appears inside the search box. The editor scrolls to show each match as you navigate.

Find Next/Previous Selection uses whatever text is currently selected as the search term, without opening the find bar. This is a quick way to jump between occurrences of a word — select it (or press Ctrl+W to select the word at the caret), then press Ctrl+F3 to jump to the next one.

Match Options

The find bar provides three toggle buttons for match options:

OptionDescription
Match Case When on, searching for "Hello" won't match "hello" or "HELLO". When off, search is case-insensitive.
Whole Word When on, matches only occur at word boundaries. Searching for "the" won't match "other" or "them".

DMEdit remembers these settings between sessions.

Search Modes

The find bar has three search mode buttons:

ModeDescription
Normal Matches the search text literally. This is the default mode.
Wildcard Use * to match any sequence of characters, and ? to match a single character. For example, err*msg matches "error_msg", "err_handler_msg", etc.
Regex Use full regular expression syntax. For example, \d{3}-\d{4} matches phone number patterns like "555-1234".

The selected search mode is remembered between sessions.

Replacing Text

Press Ctrl+H to open the find bar in replace mode. This shows a second text box for the replacement text.

ActionDescription
Replace Replace the current match and advance to the next one.
Replace All Replace every match in the document at once. For large documents, a progress indicator is shown.

Replace All is a single undo operation — press Ctrl+Z once to undo all replacements.

Incremental Search

Press Ctrl+I to start an incremental search. In this mode, you type directly into the editor and the caret jumps to the first match as you type each character. Press Escape to end the incremental search and stay at the found location, or Ctrl+I again to advance to the next match.

Find Bar Resizing

The find bar can be resized by dragging its left edge. DMEdit remembers the width between sessions.
